Interactive Prototype

任务执行继续跑，问答单独开一条侧路。

这个原型把“主执行轨”和“独立问答轨”拆开：左侧继续展示 PRD 确认或无打扰编码的主状态， 右侧只负责回答用户的临时疑问。发送问题不会自动改写 PRD、不会触发 regenerate、也不会中断当前 coding。
